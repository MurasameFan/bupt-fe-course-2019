# 第二节课资料

## 动手做一个前端界面

源码见：[lesson-2/blog-page](/lesson-2/blog-page)

### Usage

clone 下来后进入 `lesson-2/blog-page` 目录

```shell
npm install
npm start
```

有 [yarn](https://yarnpkg.com/zh-Hant/) 的同学可以这样运行：

```shell
yarn install
yarn start
```
